Aplikacja przestawia sklep z grami wiedo.

FEATURES:
- można przeprowadzić udaną transakcje
- można usunąć transakcje
- można dodać nowego użytkownika, który chce dokonać transakcji
- można dodać nowy tytuł na sklepowe pułki
